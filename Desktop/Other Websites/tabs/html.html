The Internet

People connect to the internet through cables and servers. A server is essentially what stores web pages and is connected to the internet. To find web pages, each web address has a unique IP address which can be used to find it. We connect to the internet using our computers, called clients, through an ISP (Internet Service Provider). When a client searches up something, information is broken down into several smaller 'packets'. These packets are guided to the correct IP address using routers, which are placed wherever an intersection in the cables are in order to guide the packet to its correct location. Packets work like a wrapped candy: everytime a packet goes through a router it is 'wrapped' with another layer, which tells the next router where to send it to next. When the packet is sent back to the user each 'wrapper' is unwrapped as it passes the corresponding router until it reaches the client. Browsers use a DNS (Domain Name Service) to lookup IP address' from a URL to guide the user to the correct site.

Browsers

A browser is a program on a computer that allows a user to visit sites. The user enters website addresses in a search bar and the browser returns the correct website for them. It's primary aim is to bring up websites a user searches for.

Proxy Servers

Proxy servers are special servers clients can connect to to fullfill one of these 5 main roles:

  • To obscure their client ID
  • To block mallicious traffic
  • To block certain websites
  • To log the traffic of each client
  • To improve the performance of a browser through caching websites
