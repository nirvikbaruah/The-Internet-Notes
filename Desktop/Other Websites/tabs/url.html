Protocols

Protocols are things that are agreed on between to devices so that they can communicate to each other. Setting up one is known as a handshake. They define everything a computer needs to communicate, like file format, error checking methods, data compression and through what medium data will be transferred.

HTTP is a protocol that defines how messages are transmitted and formatted over the internet.

An extension to HTTP is HTTPs, which adds an extra layer of security through SSL encryption. This means communications are more difficult to be read by intermediaries as the data is encrypted.

URLs

URLs are a way to find web content using english phrases. They represent a websites IP address. The three main parts of it are: the protocol of the website (HTTP(s)), the domain name, the type of website (.com, .sg etc.) and the file name.
