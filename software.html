Important Acronyms

  • ISP - Internet Service Provider
  • IP - Internet Protocol Address
  • MAC - Media Access Control
  • HTML - HyperText Markup Language
  • HTTP - HyperText Transfer Protocol
  • HTTPS - HyperText Transfer Protocol Secure
  • URL - Uniform Resource Locator

Internet Service Providers (ISPs)

These are internet service providers. They give the user access to the internet to allow him to connect to the network.

IP Addresses

An Internet Protocol Address is a numberical ID for each device online. Every computer, server, domain and router has a unique IP address. It consists of four three-digit numbers seperated by dots, making it 4 bytes in size. For example, 192.053.110.004 is an example of an IP address.

MAC Addresses

A Media Access Control address is a numerical ID for a device which allows you to connect to the internet. It is stored in the network interfact card of each device. It is represented as a set of 6 hexadecimal numbers, for example FE-32-11-0A-A7-09.

Cookies

Cookies are small text files stored on computers when a site is opened. They record things like passwords, logins, credit card numbers etc.
